Use

1. Creating a new experiment

Open Translog-II Supervisor, click "Project" in the menu, and then "Create project".

A pop-up window will appear:

Click on "Configure experiment" to write the text you would like your subject to work on and configure your project.

2. Running the experiments

You can open the experiment file to run in two ways:

1) Click on "User" and a pop-up window will appear:

Click "File" > "Open project" to select an experiment created in the supervisor mode. Select a file with the extension ".project" and open it.

2) The same pop-up window will appear if you open the User component directly. Click "File" > "Open project" to select an experiment created in the supervisor mode. Select a file with the extension ".project" and open it.

When you open a project, this window will appear:

Check if all the plugins you have selected are included and have the status "Ready". Then click on "Start logging" to launch the experiment.

Click on "Stop logging" when you finish and a pop-up window will appear.

Remember to save the logfile.

3. Replaying the experiment

Click "Replay" > "Open log file" to open a file generated by a user.

A file enabling you to replay all the user's activities will open.

Clicking on "Plot" will enable you to select the data you want to see on the screen.

The following panel contains play, pause, stop, forward and rewind buttons.

It is also possible to select the speed you want to replay the file at, by clicking on "Speed" and choosing the percentage from the dropdown.

"Go: 0000.000 ms" enables you to go to a specified point in your recording.
