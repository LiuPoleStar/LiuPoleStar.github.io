Introduction

Translog-II is a program to record and study human reading and writing processes on a computer. It is an instrument to acquire objective, digital data of human translation processes. Like their predecessors, Translog 2000 and Translog 2006, Translog-II also consists of two main components: Translog-II Supervisor is used to create a project file and to replay recorded sessions. Translog-II User is used to run text production experiments (a user reads, writes, or translates a text). Translog-II produces log files that contain user activity data of the reading, writing, and translation processes, and which can be evaluated by external tools (see TPR-DB).
